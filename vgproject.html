VASO Group Web Redesign


Redesigning the VASO Group website to more closely align with brand values and identity.

UX Designer

In-house Project

3 months (2022-2023)

Figma, WordPress, Elementor

Project Overview


Context

VASO Group, a Dallas-based eCommerce brand agency, wanted to revamp their website to incorporate modern trends with the company's core values: integrity, personal growth, collaboration, sense of urgency, simplicity, and communication.

Brief

  • Incorporate modern design trends into the company's brand identity to reflect core values.
  • Revamp design elements to appear more "sleek."

Design Approach

Target Users

  • VASO Group site visitors
  • New and existing clients of the business

Methods

  • Competitive Analysis
  • Wireframe

Design Process

Design Goals

01

Add new service pages to existing collection

02

Tweak website design elements to be more sleek and modern

03

Utilize both full-width and 2-column layouts to aid skimming

04

Add more graphics to replicate competitors

Prototype

Hi-fi Wireframes

Design Decisions

Q: How to evoke the "sleek" look and feel?

A: Incorporate 3D graphics and highlighting darker shades in the company's color palette

I chose a bold color palette with cold-toned shades that evoke feelings of professionalism, incorporated certain elements of the logo for consistent branding, and used 3D abstract elements (from Envato Elements) to add to the technology look.

Fully Responsive

Results

The proposed design changes were approved and developed on WordPress, built with Elementor web builder.
